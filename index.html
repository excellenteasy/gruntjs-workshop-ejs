Einrichten der Workshop App

Wir clonen uns die App mit git auf den eigenen Rechner… 

git clone git@github.com:excellenteasy/gruntjs-workshop-ejs.git

…oder forken sie hier und clonen dann. 

git clone git@github.com:deinGithubUsername/gruntjs-workshop-ejs.git

Zum Abschluss wechseln wir in die neu angelegte Repository. 

cd gruntjs-workshop-ejs

Tipp: Eigenen Fortschritt speichern


  

    Es ist immer gut wenn man den Überblick über den eigenen Fortschritt behält.
    Daher empfiehlt es sich –  wie oben vorgeschlagen – die Repository zu forken und eigene Änderungen zu speichern.
  


  

    Legen Sie dazu, nachdem Sie einen Schritt ausgecheckt haben, einen eigenen Branch für diese Aufgabe an…
  

  
    
git checkout -b my-step-00

  
  

    …und comitten, sowie pushen Sie ihre Änderungen.
  

  
    
git add --all

    
git push origin my-step-00
